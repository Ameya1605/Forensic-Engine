# Project: Forensic Engine (PREE)

An advanced AI forensic analyzer designed to reverse-engineer AI-generated text, reconstruct original prompts, and estimate system parameters.

## Core Identity
- **Vision**: To be the industry standard for LLM forensic analysis.
- **Brand**: High-tech, clinical, precise. Dark mode with glassmorphism accents.
- **Core Value**: Providing transparency into the "black box" of LLM-generated content.

## Technical Foundation
- **Frontend**: Next.js (TypeScript) with Vanilla CSS.
- **Backend**: FastAPI (Python).
- **Architecture**: Modular forensic engine with a central API coordinator.

## Scope
- Lexical Tropes Detection
- Structural Constraint Mapping
- Parameter Estimation (Temp, Top-P)
- Prompt Reconstruction
- Image Origin Detection (AI vs Human)
- Visual Artifact Analysis (Gemini Vision)
- EXIF Metadata Forensics
